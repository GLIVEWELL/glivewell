#### 应用场景

GLIVEWELL™方案可以应用于云端和边缘端，同时也可以通过通用PC构建家庭数据中心端。

###### 云端和边缘端

```
+------------------+                     +------------------+
|  云端 GLIVEWELL™  |                     | 边缘端 GLIVEWELL™ |
|     (Cloud)      |---------------------|     (Edge)       |
+------------------+                     +------------------+
```

###### 家庭数据中心（based PC）

```
+------------------+                     +----------------------+
|    Internet      |                     | 家庭数据中心GLIVEWELL™  |
|                  |---------------------|     (Edge)            |
+------------------+                     +-----------------------+
```
